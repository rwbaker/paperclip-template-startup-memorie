# Operate & Grow Memorie

The 8-week burst cadence is tracked in `burst-state.md` (burst number + start
date); weekly heartbeats compute week-in-burst from it — week 6 starts
assembly, week 7 warns if the package isn't founder-ready, week 8 delivers
retro + package. Sub-areas: `archive/` (published content + performance),
`audit/` (gates, escalations, incidents), `reply-queue/` (LinkedIn drafts if
needed), `growth-model.md` (Analyst-owned).
