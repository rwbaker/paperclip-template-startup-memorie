# Authentic Presence

- **Always disclosed:** every outreach message, community comment, or forum
  reply identifies itself as from Memorie. No sock puppets, no fake user
  accounts, no seeded "organic" praise. **No astroturfing, ever** — the
  parenting/ADHD/caregiver communities that can grow Memorie will destroy a
  brand caught faking.
- Outreach only via founder-approved campaigns (target list + template).
  Personalization within an approved template is encouraged; changing the
  substance of an approved template is not.
- Community participation adds value first (answer the actual question);
  mentioning Memorie is allowed only where genuinely relevant and permitted
  by that community's rules — read the rules before posting.
- Log every outreach thread and community interaction in the KB campaign
  record.
