# Evidence Standards

1. **Cite or it doesn't count.** Every demand/market claim needs a verifiable citation: source URL + a short verbatim quote + a count (e.g. "47 complaints"). Uncited assertions contribute 0 to the score.
2. **Corroborate.** A high score on Demand or Willingness-to-pay requires ≥2 **independent** sources (not two links to the same thread).
3. **Adversarial pass (mandatory).** Before finalizing any brief, write the strongest case for why this idea fails: who already tried and died, why demand may be shallow, why incumbents win. Only then state whether the thesis survives. Agents default to cheerleading — this step is the counterweight.
4. **Freshness.** Re-verify a finalist's competition section immediately before surfacing it for a build decision; a brief can go stale between validation and the founder's review.
